# 사용자 정책

## 회원가입 입력 규칙

### 아이디 (userId)

| 항목 | 규칙 |
|------|------|
| 허용 문자 | 영문(대·소문자) + 숫자만 허용. 한글·공백·특수문자 불가 |
| 길이 | 최대 50자 |
| 정규식 | `^[a-zA-Z0-9]+$` |
| 오류 메시지 | "아이디는 영문과 숫자만 사용할 수 있습니다." |

### 비밀번호 (password)

| 항목 | 규칙 |
|------|------|
| 필수 조합 | 영문 1자 이상 + 숫자 1자 이상 + 특수문자 1자 이상 |
| 허용 특수문자 | `!@#$%^&*()-_=+[]{};\:'",.<>/?\\|` |
| 길이 | 최소 8자 |
| 정규식 | `^(?=.*[a-zA-Z])(?=.*[0-9])(?=.*[!@#$%^&*()\-_=+\[\]{};:'\",.<>/?\\|]).+$` |
| 오류 메시지 | "비밀번호는 영문, 숫자, 특수문자를 각각 1자 이상 포함해야 합니다." |

### 이름 (userNm)

| 항목 | 규칙 |
|------|------|
| 필수 여부 | 선택 입력 |
| 길이 | 최대 100자 |

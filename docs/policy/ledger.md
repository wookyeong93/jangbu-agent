# 장부 정책

## 장부 등록 규칙
- 항목: 매입 / 매출 / 지출 3종만 허용
- 금액: 양수 정수만 (소수점 불가)
- 거래일: 기본값 오늘, 사용자 수정 가능
- 거래명 : 필수 값이 아니므로 미입력 가능 
- 수정 가능 필드: 항목 / 거래일 / 금액 / 거래명 

## 삭제 정책
- user : ledger = 1 : N, ON DELETE CASCADE
- user 삭제 시 해당 장부 전체 삭제 (보존 안 함 → ADR 참조)

## 월별 KPI (파생, 미저장)
- 총매입/총매출/총지출: 해당 월 항목별 amount 합
- 순이익 = 매출 − 매입 − 지출
- 마진율(%) = (매출 − 매입) / 매입 × 100, 매입=0 이면 0
